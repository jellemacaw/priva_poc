Basic usage

The colors as defined in the online styleguide are codified by their color names and then abstracted to the color definitions as seen above. These variable names are imported and used in the custom bootstrap variables file so Bootstrap still uses its own internal variable names. E.g.:

_color-names.scss         _color-theme.scss         _bootstrap-custom-variables.scss

$color--blue-dianne -->   $color--primary -->       $brand-primary
