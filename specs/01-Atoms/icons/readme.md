# Icon Spec

There are several different types of icons.

## Social media icons
As used in the footer

```code
<a href="#" class="fa fa-home fa-fw" aria-hidden="true"></a>
```
### Example
```example
<div class="icons-sm-spec"></div>
<script>
	var target = ".icons-sm-spec";
	var block = "variants/socialmedia.html";
	$(target).load(block);
</script>
```
